# Movie Explore

## About

- Stack:
  - mySQL
  - Express
  - Nodejs
  - ReactJS


## Installation

```
npm install

npm run build
```

### Database needs tcp/ip connection

```
screen

ssh -L 3306:127.0.0.1:3306 studentweb.comminfo.rutgers.edu -l<user-name>
```


### Run the app

`dev`
```
npm run server
```
`production`
```
npm run production
```


#### Open your browser to localhost:9000
